    Class ItemEventHandlerAttribute

    Used to mark a method as an item event handler.

    Before item event signature: bool OnBeforeItemEvent(ItemEvent e);

    After item event signature: void OnAfterItemEvent(ItemEvent e);

    To stop Business One from processing the event return false on the before event.
