    Interface IConnection

    Represents a connection to SAP Business One. A connection can be obtained from the ConnectionFactory.

    Namespace:B1PP.Connections
    Assembly:B1PP.Framework.dll
    Syntax
    public interface IConnection
    Examples
    How to perform a standard (UI and DI API) connection to SAP Business One. 
    IConnection connection = ConnectionFactory.CreateStandardConnection();
connection.Connect();
// Your code
connection.Disconnect();
